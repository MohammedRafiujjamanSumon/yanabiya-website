How to Log In

Accessing the Admin Panel

http://localhost:5173/admin/login
1

Open the login page

Go to the URL above in your browser, or click the ⚙ Admin Login link at the bottom-right of the website footer.

2

Enter your credentials

Email: admin@yanabiyagroup.com   /   Password: YanabiyaAdmin@2024

3

Click "Sign In"

You will be taken to the Admin Dashboard. Your session stays active for 7 days — no need to log in again unless you clear your browser.

Tip: The website must be running locally (npm run start) before you can log in. The API server runs on port 4000 and the website on port 5173.

Page Builder

/admin/pages

The Page Builder lets you create completely new custom pages on the website without any coding. Each page is made up of sections that you add, edit, reorder, and delete.

Creating a New Page
1

Go to Page Builder → click "+ New Page"

Enter the page title (e.g., "Investment Opportunities"). The URL slug is auto-generated.

2

Set page status

Choose Draft (only you can see it) or Published (live on the website).

3

Click "Create Page"

You will be taken to the Page Editor where you can add sections.

Adding Sections
1

Click "+ Add Section"

A grid of 12 section types appears. Choose the type you want.

2

Edit the section

Click the expand (▼) arrow on the section card to open its editor and fill in content.

3

Reorder / hide / delete

Use ▲▼ arrows to reorder. Eye icon to show/hide. Trash icon to delete.

4

Save Sections

Click Save Sections button. Published page is live at yoursite.com/p/slug

12 Available Section Types

H Hero

  • Big banner with title, subtitle, button, background image

T Text

  • Heading and paragraph text block

C Cards

  • Grid of feature cards with icon, title, text

S Stats

  • Key numbers / statistics display

A CTA

  • Call-to-action block with button

F FAQ

  • Expandable question & answer list

Q Testimonials

  • Quote / review carousel

P Team

  • People profile cards with photo

✉ Contact

  • Contact form or details block

G Gallery

  • Photo / image grid gallery

— Spacer

  • Empty vertical space divider

<> HTML

  • Custom raw HTML code block
How to view your page: Click the eye icon (View) next to any page in the list, or go to http://localhost:5173/p/your-page-slug

Media Library

/admin/media

The central place to upload, organize, and manage all images and videos used on the website. Every image you upload here gets a URL that can be pasted into any editor field.

↑ How to Upload

  • Select a folder (logo, hero, partners, services, pages, misc…)
  • Drag & drop your file onto the upload zone
  • OR click the zone to pick a file from your computer
  • A progress bar shows the upload status
  • The file appears in the grid when done

⊡ How to Use an Uploaded Image

  • Hover over any file card
  • Click the Copy icon (clipboard)
  • The full image URL is copied
  • Paste the URL into any image field in the editors

F Folders

  • logo — brand assets
  • hero — banner images
  • partners — partner logos
  • services — service icons
  • countries — country photos
  • leadership — profile photos
  • pages — page images
  • misc — everything else

⊞ Filter & Search

  • Search by filename
  • Filter: All / Images / Videos
  • Filter by folder tab
  • Click Refresh to reload

✗ Delete a File

  • Hover over the file card
  • Click the Trash icon
  • Click again to confirm
  • File is permanently deleted
Supported formats: JPG, PNG, SVG, WebP, GIF (images) — MP4, WebM, MOV, AVI, MKV (videos up to 200 MB)
